# Medical Center Platform

The **Medical Center Platform** is a Python-based application for managing **appointments and patients** in a medical center.  
Built with **Flask** and **PostgreSQL**, it demonstrates backend development, modular design, and MVC architecture.

## Features
- Create, view, and manage appointments for patients.    
- Modular backend for maintainability and scalability.

## Tech Stack
- **Python (Flask)**  
- **PostgreSQL**
- **HTML/CSS/JS**

Learning Outcomes
Flask backend development and MVC design.
PostgreSQL integration.
User roles and notifications logic.
Modular, maintainable application structure.
